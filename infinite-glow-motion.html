💡 How To Use

All effects run infinitely and continuously - no hover needed! Just add the animation to any element.

📝 Basic Template

/* 1. Define the animation with @keyframes */ @keyframes pulseGlow { 0%, 100% { box-shadow: 0 0 20px rgba(100, 200, 255, 0.3); } 50% { box-shadow: 0 0 60px rgba(100, 200, 255, 0.8); } } /* 2. Apply to any element */ .element { animation: pulseGlow 3s ease-in-out infinite; background: #1a1a2e; border-radius: 12px; padding: 60px 40px; }

🎨 Customize These Values

Duration: Change 3s to 2s (faster) or 5s (slower)

Colors: Replace rgba(100, 200, 255) with your color (blue, red, green, etc.)

Intensity: Change 0.8 opacity to 1 for stronger glow

Size: Change 60px to larger/smaller glow spread

⚡ Quick Codes

/* PULSING */ animation: pulseGlow 3s ease-in-out infinite; /* EXPANDING */ animation: expandGlow 2.5s ease-in-out infinite; /* COLOR SHIFT */ animation: colorShiftGlow 5s linear infinite; /* BREATHING (with scale) */ animation: breathingGlow 3s ease-in-out infinite; /* RAINBOW */ animation: rainbowGlow 4s linear infinite;
